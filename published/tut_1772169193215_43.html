How to Manage Your API Tokens on Replicate

This tutorial will guide you through navigating to your API token settings on the Replicate dashboard. By following these steps, you can easily access the options to manage your individual API tokens.

Before you start

  • A Replicate account
  • Logged in to your Replicate account

1 Start on the Replicate dashboard.

Dashboard – Replicate

2 Click your profile picture or username 'easyearl1' to open the account menu.

Dashboard – Replicate

3 In the account menu, click 'API tokens' to navigate to your API token settings.

Dashboard – Replicate

4 On the 'API tokens' page, click the action icon next to the API token you wish to manage.
